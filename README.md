# 3ra Entrega – Coderhouse

Este es mi 3ra entrega del curso. La idea fue armar una web responsive usando HTML, SASS y Bootstrap. La navegación incluye un menú hamburguesa para mobile y los estilos están organizados con SASS.

---

## ¿Qué tiene el proyecto?

- Páginas: inicio, productos, servicios, contacto y nosotros.  
- Carpeta `assets/` con imágenes, CSS y SCSS.  
- SASS organizado en módulos (variables, mixins, layout, componentes, etc.).  
- `style.scss` se compila a `style.css` usando Live Sass Compiler.

---

## Tecnologías usadas

HTML – SASS – Bootstrap – Flexbox – GitHub
